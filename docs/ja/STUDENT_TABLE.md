# 学生向け表

`rugby-value table` は、モデルの全情報から次の列だけを抽出します。

| 列 | 意味 |
|---|---|
| エリア | ピッチ縦方向の位置 |
| 横位置 | 左・中央・右 |
| 起点 | ラインアウト、スクラム等 |
| EPV | そのポゼッション開始からのネット得点期待値 |
| 得点確率 | 自チームが何らかの得点をする確率 |
| トライ確率 | 自チームがトライする確率 |
| 無得点確率 | ポゼッションが得点なしで終了する確率 |
| 標本数 | 当該開始状態の観測数 |
| 信頼区間 | 試合単位ブートストラップによるEPV区間 |

この表は「実際のトップリーグで、その起点から平均的にどれだけ価値があったか」を示します。個別のパス・キャリー・キックの最適選択までは示しません。
